A jQuery plugin to make your elements look like a jigsawGithub

Features

jJigsaw is a jQuery plugin to create a jigsaw effect on an element,which is useful for stylishness.

It mainly uses CSS border tricks and position tricks to achieve the effect, without using any image, which enables us to create this kind of effect of any size.

It's a tiny library with minified JavaScript file of 6KB.

Hello World!

To use jCorner, the following files should always be included.

        <script src="jquery.min.js" type="text/javascript"></script>
        <script src="jJigsaw.js" type="text/javascript"></script>

Suppose we want to add this jigsaw effect to an element with id jigsaw in the HTML, jJigsaw makes it as simple as:

HTML:

        <div id="jigsaw"></div>

JavaScript:

        $('#paper').jJigsaw();

Documentation

There're 4 parameters to set with jJigsaw, none of which is compulsory. Default values are:

            bgcolor:"#ffffff",
            bdcolor:"#cccccc",
            bdwidth:5,
            holerate:8

If your web page had set the background-color property of body element,jJigsaw will set the same color value of the bgcolor parameter automatically.

Example:

            $("#jigsaw").jJigsaw({
            holerate:7,
            bdcolor:"#AAAAAA"
            });
            $("#jigsaw").jJigsaw({
            holerate:5,
            bdcolor:"#cccccc",
            bdwidth:3
            });

bgcolor

In order to make your elements have this jigsaw effect,you had better ensure the bgcolor parameter is set to be the same value of your elements' container.

bdcolor

This parameter defines your elements' shadow effect,bdcolor is set to be the shadow's color.

bdwidth

You may want to control the size of shadow in pixels,that's exactly what this parameter do.

holerate

Well,I have to admit that it's hard for me to explain the usage of this parameter.The larger it is , the smaller the holes of your jigsaw it will be.

P.S.

If your element have child element,you had better set some CSS properties such as margin for a better effect. Besides,if you have set the background-image property for your element,please do as follows: 

            #jigsaw{
                background:url(bg.jpg) no-repeat 0 0;
                background-size:100% 100%;
            }
